Trading Journal

Your personal trading journal with Telegram bot integration, analytics, and cTrader support.

📚 View on GitHub 🤖 Open Bot

✨ Features

📱 Telegram Bot - Log trades conversationally
📈 Chart Generation - Automatic candlestick charts with entry/SL/TP
🔄 cTrader Integration - Auto-import trades from your broker
📊 Advanced Analytics - Win rate, Sharpe ratio, Monte Carlo simulation
🎯 Weekly Goals - Track adherence to trading plans
🧠 Psychology Tracking - Mood, stress, confidence metrics
💼 Multi-Account - Manage multiple prop firm accounts
✅ Repository: Deployed to GitHub
🔗 URL: https://github.com/brusnyak/trading-journal
📱 Telegram Mini App:
This page serves as the landing page for your trading journal.

To use as Mini App:
1. Update your bot's .env file:
WEBAPP_URL=https://brusnyak.github.io/trading-journal
2. Restart the bot
3. Send /mini command to your bot
4. Tap the button to open the app inside Telegram
