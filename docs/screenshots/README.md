# Screenshots

This folder contains screenshots used in the PROJECT.md documentation.

## How to Add Screenshots

1. Take screenshots of your trading journal features
2. Save them with descriptive names in this folder
3. Reference them in PROJECT.md using relative paths

## Screenshot Naming Convention

Use descriptive, kebab-case names:
- `dashboard.png` - Main dashboard overview
- `direction-analysis.png` - Trade direction analysis
- `wins-losses.png` - Wins vs losses breakdown
- `risk-reward.png` - Risk-reward analysis
- `time-analytics.png` - Time-based analytics
- `monte-carlo.png` - Monte Carlo simulation
- `trade-replay.png` - Trade replay feature
- `weekly-review.png` - Weekly review page
- `psychology-tracking.png` - Emotional tracking
- `telegram-bot.png` - Telegram bot interface

## Markdown Syntax

To reference screenshots in PROJECT.md:

```markdown
![Alt Text](docs/screenshots/filename.png)
*Optional caption text*
```

Example:
```markdown
![Dashboard Overview](docs/screenshots/dashboard.png)
*Main dashboard showing performance metrics and equity curve*
```

## Image Guidelines

- **Format**: PNG or JPG
- **Size**: Max 2MB per image
- **Resolution**: 1920x1080 or smaller
- **Quality**: High quality, clear text
- **Privacy**: Remove any sensitive account information
